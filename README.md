# 任务管理系统项目
## 实现功能介绍
使用umi搭建React应用，任务管理系统（使用Class组件实现）
包括以下基本功能
* 一、任务管理
  * 属性：标题、描述、类型、优先级、状态、作者、指派人（除描述外，全部必填）
  * 场景：列表、创建、编辑更新、删除、详情查看
  * 主页面：
  任务列表、右侧查看详情、弹出窗口新建/编辑
* 二、属性后台管理
  * 类型管理
  属性：名称；必填，唯一
  场景：列表方式的增删改
  * 优先级管理，同类型操作
  * 状态，同上
  * 人员，同上
* 三、统计后台管理
  * 各成员的各状态任务单量
